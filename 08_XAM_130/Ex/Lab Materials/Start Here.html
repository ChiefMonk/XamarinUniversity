Overview

This session shows how to create Xamarin.Forms user interfaces declaratively with XAML. You will also see how to write code that interacts with your XAML UI so you can retrieve user input and display output.

Before you begin

This is a Xamarin.Forms class. To complete the coding exercises, you will need a Xamarin development environment with either a simulator/emulator or a physical device. Please read the setup requirements if you need guidance installing the Xamarin development environment.

Objectives

  1. XAML syntax
  2. Adding Behavior to XAML-based UI
  3. Advanced XAML

Lecture

XAM130 - XAML in Xamarin.Forms.

Exercises

  1. Create a XAML-based version of Calculator
    Exercise
    Convert the UI of an existing Xamarin.Forms application to use XAML.
  2. Adding Behavior to XAML Calculator
    Exercise
    Connect the XAML-based UI to the code that implements the app's behavior.
  3. Cleanup the XAML code and tailor the UI to the platform
    Exercise
    Use static data to eliminate repeated values in the XAML. Fine-tune the UI with platform-specific values.
